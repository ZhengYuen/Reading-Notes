第一章 幸福是目的，成功是手段
人生是一条河
  1. 幸福感的本源：基因的传承和影响力。
  2. 人生是一条河，每个人总希望自己这条河能够更深一点更宽一点，更长一点。更深一点要靠个人的修行和对世界的理解，更宽一点要和志同道合的人在一起共同做一些事情。
这个世界没有欠你什么
  1. 任何社会都是分层的，稍微好一点的社会只不过是有一个上下层的通道，让人员可以流动而已。
  2. 庆幸的是，绝大部分人在一生中的大部分时间都是处在松懈状态，而达不到 80%的努力程度。
  3. 对那些试图在金字塔上爬几个台阶的人来说，要做的绝不是抱怨社会的不公平，而是需要付出足够的努力，同时把注意力放到最该关注的事情上去。
人生最重要的投资
  1. 对于年轻人来说，对自己的投资和在职业上的进步，远比在股市上捞点钱或者向父母借钱买一套房子更为重要也更为靠得住。
  2. (J. P. Morgan)一旦婚姻投资得当，你的事业也将随之达到高峰。假如把婚姻视为儿戏，草率决定，随之而来的惩罚将是离婚、精神痛苦，以及存款金额的锐减。
  3. (J. P. Morgan)好妻子的基本要求：迷人、有气质、聪明，不搬弄是非而且性格好。
  4. (J. P. Morgan)女生择偶：找一个比自己更优秀的人；趁着年轻的时候将自己嫁出去。
  5. 对孩子，要给他们充分的自由，包括犯错误的自由。
给男生的建议
  1. 有且只有聪明人会欣赏聪明人。
  2. 如果一个人的心智不成熟或者两个人的价值观和文化习惯完全不匹配，婚姻是难以长久的。
  3. 男生应该有足够的独立性，并且承担对自己未来家庭的义务。在一个完美的婚姻中，双方都要明白关系的亲密程度和重要性依次是：夫妻优于子女，更优于双方的父母。
  4. (有些)美德，只不过是农耕文明的产物而已。
  5. 一个人，特别是年轻的时候，可塑性很重要。
给女生的建议
  1. 大部分时候，爱情和婚姻对女生的重要性超过它们对男生的重要性。
  2. 完美的爱情不意味着完美的婚姻(《梁祝》《罗密欧与朱丽叶》《神曲》）
  3. 婚姻和爱情是不同的。恋爱是激情，是化学物质分泌所带来的愉悦；婚姻则是由两个人一同构造一个舒适的共同体，在那个共同体中，双方都将收益。
  4. 一段完美的爱情，仅仅是一个好的开局。靠化学物质维持的激情不能持久，降温后是否能变成一份温情、一种亲情关系。这个过程不仅需要双方努力，而且需要对方是一个合适的人，否则在努力也没有用。

第二章 人生需要做减法
不做选择的幸福
  1. 印度人缺乏选择的状态，以及不选择而产生的幸福感和成就感，帮助了他们的精英在公司里取得成功。
  2. 人的幸福感和成就感的取得，不是来回选择的结果，而是在没有多少选择的时候深度经营的结果。
  3. 选择太多，导致人太浮躁，不愿意长期在一个地方、一个领域努力经营，而做到顶级是需要时间的。我们把太多的精力花在了选择上，而不是经营上，导致难以精进。
做人与作诗：我们需要林黛玉
  1. 形容之美终不如举止优雅、腹有诗书。
  2. 在现实生活中，我们需要学会做人，光有作诗的性格不见容于世，这一点没有疑问。但是世界上不能缺少林黛玉这样以生命作诗的人，否则一个社会就是一个庸俗的社会，一个国家就是一个庸俗的国家。(屈原、李白、李商隐、贝多芬、托尔斯泰、梵高、海明威、雪莱、拜伦、济慈)
  3. 很多键盘侠脑子里想的不过是有房有车的生活，所抱怨的不过是自己还没有那两样东西，这些人多少玷污了“左派”二字。至于那些天天在媒体上发声、把平等挂载嘴边的精英，如果真的同情“难民”，不妨把他们安置在自己家中，而不要把他们安置在不欢迎“难民”的社区。
  4. 上帝赋予的特殊天赋是机器无法取代的：艺术的创造力和想象力；梦想和浪漫的情怀。
  5. 人生不仅要做人，也要作诗。
西瓜与芝麻
  1. (一些行为)不仅在于时间利用得非常没有效率，更糟糕的是他们渐渐养成了习惯于非常低层次的追求。
生也有涯，知也无涯
  1. 我(作者)做事的诀窍：少做事情，甚至不做事情。
  2. 我(作者)的优点：能跳出思维定式，换一个角度来判断一件事情的重要性；其次，敢于舍弃。
  3. 很多事情不做其实也无关大体，就不要去做它们，这样你就不会每天忙忙碌碌了。
  4. 想要换取更多的时间，就必须牺牲很多经济利益。但是生命是由有限的时间构成的，而钱超过一定程度后，其实并不重要了。
我们一定比十八世纪的人过得好吗？
  1. 贵族作为一个阶级不存在了，但是他们的精神和生活方式依旧存在。
  2. 贵族安身立命的三个根本：军事上的责任、维护地区治安的义务和社会活动时的体面。
  3. 参加军事和政务--->荣誉感和责任感--->重承诺的习惯
  4. 贵族举止上最重要的是在自然危险面前的淡定。
  5. 在十八世纪，生活节奏远远没有今天这么快，因此贵族们的生活讲究从容、自律和优雅。可是这些和钱的关系不太，这种令人神往的气质和自信其实来源于他们的内心在用责任和荣誉对自己进行约束，对完则展现出从容和优雅。这些是任何时候幸福生活的根本。
  6. 幸福的来源，除了基因的传承和成就的影响力之外，有三个维度：爱情和婚姻、对未来的期望、生活的态度。
  7. 今天的人并不太担心物质的匮乏，如果我们每天在出门前能想到“责任”“荣誉”“从容”“优雅”“镇定”就能比18世纪的贵族过得更好。

第六章 职场误区与破解
第一份工作不要太在乎工资
  1. 对于公司非常重要的岗位，公司会开出一个比市场价高一倍的薪水，这样就由公司来挑选最好的人，而不是让最好的人在几家公司中作比较。用同样的方法选择会计师和律师等专业人士。
  2. 为了引入一流的人，而且让他们安心为你做很长时间的事情，最好的办法就是给他一个高于预期的待遇。
  3. 人的第一份工作必须能帮助你再10年后挣到同龄人、同班同学3~5倍的收入，这样你才能在北京买得起房子。
  4. 人的第一份工作必须能让你快速地成长，养成良好的职业习惯，在最短的时间里了解全行业。
  5. 每个人都有自己的起点，自己和自己比，重要的是每过几年就能够真正上一个台阶。
  6. 无论是在中国还是在美国，在乎20%工资的人要比注重自己成长的人多，因此就给有志气的人留下了机会。
专业人士评价体系
  1. 朗道等级最核心的思想是，人和人的差距、能力和能力的差距，是数量级的差别，而不是人们想象中的一点点。
  2. 五星工程师：能独立完成所分配的工作，无需他人指导；
  3. 四星工程师：有领导能力，在工程上能把大问题化解为小问题；
  4. 三星工程师：独立带领一队人做出能为公司挣得利润的产品。--->对市场的判断和营销能力；--->自身也是好的产品经理；
  5. 二星工程师：能做出以前没有的东西；
  6. 一星工程师：开创一个产业；
职场的几个误区
  1. 工作和职业分不清；把自己当成是过客而不是主人；被语言暴力激怒后乱了章法；疏于沟通
  2. 沟通的目的不一定是为了达成一致，而是发出自己的声音。
  3. 沟通中保持对别人的尊重，但是态度要坚决明确。
  4. 工作不是为了公司或其他人，而是为了自己的职业。
  5. 注重长期效益，把看问题的周期放得长远些。
  6. 最有效的沟通是第一时间给出答案，然后补充解释。
  7. 理解对方的提问的目的，然后提供有效信息。
  8. 基层员工要抬起头，而管理者要弯下腰。
职场的进阶
  1. 0-50分靠常识；
  2. 50-90分靠先进的科学和技术；
  3. 90-100分靠艺术；

第八章 理性的投资观
金钱观
  1. 钱是上帝存在你那里的，不是给你的，你是要还的。
  2. 钱花出去才是你。
  3. 钱和任何东西都是为了让生活更加美好，而不是更加麻烦的。
  4. 钱是挣出来的，不是省出来的。挣钱的效率取决于一个人的气度。
  5. 钱是花不完的，但是可以迅速地投(投机、投资)光。
  6. 每个人的能力都是有极限的，在这个极限内努力，效果会比较好；接近了极限，效率就会打折扣；想超越极限，是枉费心机。--->是所谓不逾矩
  7. 做人做事永远要有风险意识，才能立于不败之地。
投资
  1. 用理性的态度、量化的方法看待各种事情处理各种事情。
  2. 投资的目的：发财或回报；保证财产不受损失；为了情怀。
